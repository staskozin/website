## Команды

### `dev`
Запускает приложение в режиме разработки.
```sh
npm run dev
# next dev
```

### `build`
Собирает приложение.
```sh
npm run build
# next build
```

### `start`
Запускает сайт на `localhost:3008`.
```sh
npm run start
# next start -p 3008
```
